# 主布局架构重构（顶栏 + 主区域 + 设置抽屉分层）

## 概述

将颜色模式切换与设置按钮从 `ConversationArea` 的 header 中独立，重构为**分层布局架构**：

```
┌─────────────────────────────────────────────────────┐
│ TopBar (h-12)  [主题切换] [设置按钮]               │ ← 新增顶栏
├──────────────┬──────────────────┬──────────────────┤
│              │                  │                  │
│ Sidebar      │ ConversationArea │ SettingsDrawer   │
│ (可折叠)     │ (主对话)         │ (宽度动画 0→20rem) │
│              │ (可滚动)         │ (从右弹出)       │
│              │                  │                  │
└──────────────┴──────────────────┴──────────────────┘
```

**核心变化**：
- ✅ 顶栏与主区域分离，无边框线设计
- ✅ 设置区从右侧弹出时，主对话区自动压缩
- ✅ 主对话区保留滚动条，贴近设置区边界
- ✅ 设置区支持上下滚动，隐藏滚动条视觉

## 背景与当前状态

### 当前架构

**HomePageLayout.tsx 结构**：
```jsx
<div className="flex h-screen w-screen">
  <ConversationSidebar ... />
  <div className="flex flex-1 flex-col">
    <ConversationArea 
      onSettingsToggle={handleSettingsToggle}
      onThemeToggle={() => setIsDark(...)}
      showHeader={true}  // ← 包含 h-12 的 header
      ...
    />
    {/* DocumentPanel / ArtifactPanel （可选右侧面板） */}
  </div>
  <SettingsDrawer isOpen={isSettingsOpen} ... />
</div>
```

**ConversationArea.tsx header**：
```jsx
{showHeader && (
  <header className="h-12 shrink-0 bg-surface-alt/95 backdrop-blur">
    <div className="flex h-full w-full items-center justify-between px-4 md:px-6">
      <div>
        <h1>{conversationTitle}</h1>
        {sessionMetaText && <div>{sessionMetaText}</div>}
      </div>
      <div className="flex items-center gap-2">
        <button onClick={onThemeToggle}>  ← 主题切换
          {isDark ? <Sun /> : <Moon />}
        </button>
        <button onClick={onSettingsToggle}>  ← 设置按钮
          <Settings />
        </button>
      </div>
    </div>
  </header>
)}
```

**SettingsDrawer.tsx 结构**：
```jsx
<div 
  className="shrink-0 overflow-hidden bg-surface-alt transition-[width] duration-300"
  style={{width: isOpen ? '20rem' : '0'}}
>
  <div className="flex h-full w-[20rem] flex-col">
    {/* 顶部占位与 header (h-12) 对齐 */}
    <div className="h-12 shrink-0" aria-hidden="true" />
    {/* 设置内容 */}
  </div>
</div>
```

### 问题

1. **顶栏职责混乱**：主题/设置按钮与对话标题混在同一 header 中
2. **布局不清晰**：SettingsDrawer 需要占位符与 ConversationArea header 对齐，增加维护成本
3. **灵活性受限**：难以独立控制顶栏样式或未来添加其他全局控件
4. **分层设计缺失**：顶栏、主区域、设置区的职责边界模糊

## 设计方案

### 1. 顶栏组件（TopBar）

**职责**：
- 显示会话标题与元信息（左侧）
- 容纳主题切换与设置按钮（右侧）
- 作为固定的顶部导航层

**样式要点**：
- `h-12` 固定高度，`shrink-0` 不参与 flex 压缩
- `bg-surface-alt/95 backdrop-blur`（与 ConversationArea 保持一致）
- **无边框线**（移除 `border-b` 或 `border-r`）
- 内部布局：`flex items-center justify-between px-4 md:px-6`

### 2. ConversationArea 调整

**改动**：
- 移除 `showHeader` 逻辑（或设为 `false`）
- 去掉顶部 h-12 header
- 消息列表直接从容器顶部开始
- 保留边框样式（仅在文档区存在时调整）

**滚动条处理**：
- 主对话区 `overflow-y-auto`，滚动条宽度保持一致（通常 8-12px）
- 当 SettingsDrawer 打开时，滚动条自动贴在两区域交界处

### 3. SettingsDrawer 简化

**改动**：
- 移除顶部 `h-12 shrink-0` 占位符（不再需要与 header 对齐）
- 内容直接从顶部开始
- 宽度动画保持不变：`0 → 20rem`，`transition-[width] duration-300`

**滚动条隐藏**：
- 保持 `overflow-y-auto` 支持滚动
- 使用 `.settings-scrollbar-hidden` 隐藏可视滚动条
- 用户仍可触摸滚动或使用鼠标滚轮

### 4. 布局嵌套调整

**新的 HomePageLayout 结构**：
```jsx
<div className="flex h-screen w-screen flex-col">
  {/* 顶栏层 */}
  <TopBar
    conversationTitle={conversationTitle}
    sessionMetaText={sessionMetaText}
    isDark={isDark}
    onThemeToggle={() => setIsDark(...)}
    onSettingsToggle={handleSettingsToggle}
  />

  {/* 主容器层 (flex-1) */}
  <div className="flex min-w-0 flex-1 overflow-hidden">
    <Sidebar ... />
    
    <div className="flex min-w-0 flex-1 flex-col overflow-hidden">
      <ConversationArea
        showHeader={false}  ← 关闭 header
        ...
      />
      {/* DocumentPanel / ArtifactPanel */}
    </div>

    {/* 设置抽屉 (宽度动画) */}
    <SettingsDrawer isOpen={isSettingsOpen} ... />
  </div>
</div>
```

## 实现步骤

### Phase 1：提取 TopBar 组件

#### 1.1 创建 TopBar 组件

**文件**：`frontend/src/app/home/components/TopBar.tsx`

```tsx
interface TopBarProps {
  conversationTitle: string
  sessionMetaText?: string | null
  isDark: boolean
  onThemeToggle: () => void
  onSettingsToggle: () => void
}

export function TopBar({
  conversationTitle,
  sessionMetaText = null,
  isDark,
  onThemeToggle,
  onSettingsToggle,
}: TopBarProps) {
  return (
    <header className="h-12 shrink-0 bg-surface-alt/95 backdrop-blur">
      <div className="flex h-full w-full items-center justify-between px-4 md:px-6">
        <div className="min-w-0 flex items-center gap-3">
          <h1 className="line-clamp-1 text-sm font-medium text-text-primary">
            {conversationTitle}
          </h1>
          {sessionMetaText && (
            <div className="shrink-0 text-xs text-text-muted">
              {sessionMetaText}
            </div>
          )}
        </div>
        <div className="flex items-center gap-2">
          <button
            type="button"
            onClick={onThemeToggle}
            className={[
              'flex items-center justify-center',
              'size-9 rounded-lg',
              'text-text-secondary',
              'transition-colors hover:bg-hover hover:text-text-primary',
            ].join(' ')}
            aria-label={isDark ? '切换到亮色模式' : '切换到暗色模式'}
          >
            {isDark ? <Sun className="size-5" /> : <Moon className="size-5" />}
          </button>
          <button
            type="button"
            onClick={(e) => {
              e.stopPropagation()
              onSettingsToggle()
            }}
            className={[
              'flex items-center justify-center',
              'size-9 rounded-lg',
              'text-text-secondary',
              'transition-colors hover:bg-hover hover:text-text-primary',
            ].join(' ')}
            aria-label="打开设置"
          >
            <Settings className="size-5" />
          </button>
        </div>
      </div>
    </header>
  )
}
```

### Phase 2：重组 HomePageLayout

#### 2.1 调整主容器结构

```tsx
export function HomePageLayout() {
  // ... 所有 state 保持不变

  const sidebarItems = useMemo(() => sessionItems.map(toConversationItem), [sessionItems])
  const sessionMetaText = currentSessionId ? `会话 ID: ${currentSessionId}` : null
  const showDocumentWorkspace = Boolean(openDocument)

  return (
    <div
      className={[
        'flex h-screen w-screen flex-col',  ← 改为 flex-col
        'bg-surface-alt text-text-primary font-sans',
        'transition-colors duration-300',
      ].join(' ')}
    >
      {/* 顶栏层 */}
      <TopBar
        conversationTitle={conversationTitle}
        sessionMetaText={sessionMetaText}
        isDark={isDark}
        onThemeToggle={() => setIsDark((prev) => !prev)}
        onSettingsToggle={handleSettingsToggle}
      />

      {/* 主容器层 (flex-1 flex 布局) */}
      <div className="flex min-w-0 flex-1 overflow-hidden">
        <ConversationSidebar ... />
        
        <div className="flex min-w-0 flex-1 flex-col overflow-hidden">
          {/* 会话错误提示（可选） */}
          {sessionError && (
            <div className="shrink-0 border-b border-border bg-surface px-4 py-2 text-xs text-text-muted">
              {sessionError}
            </div>
          )}

          {/* 会话概览或主对话区 */}
          {activeView === 'sessions' ? (
            <ChatsOverview ... />
          ) : (
            <div className="flex min-h-0 flex-1 overflow-hidden">
              {/* 主对话区 + 文档区 */}
              <div className="flex min-h-0 min-w-0 flex-1 overflow-hidden">
                <ConversationArea
                  showHeader={false}  ← 关闭 header
                  ...
                />
              </div>

              {showDocumentWorkspace && openDocument && (
                <>
                  {/* 分割线 */}
                  <div role="separator" ... />
                  
                  {/* 文档面板 */}
                  {openDocument.kind === 'attachment' ? (
                    <DocumentPanel ... />
                  ) : (
                    <ArtifactPanel ... />
                  )}
                </>
              )}
            </div>
          )}
        </div>

        {/* 设置抽屉 */}
        <SettingsDrawer
          isOpen={isSettingsOpen}
          onClose={() => setIsSettingsOpen(false)}
          ...
        />
      </div>
    </div>
  )
}
```

### Phase 3：样式调整

#### 3.1 移除主容器的 padding

```tsx

// 新的：

```

#### 3.2 验证滚动条贴边效果

确保 ConversationArea 的滚动条在 SettingsDrawer 打开时自然贴在交界处：
- ConversationArea: `overflow-y-auto` + `flex-1 min-h-0`
- SettingsDrawer: `width: isOpen ? '20rem' : '0'` + `transition-[width]`

Flexbox 自动处理宽度分配，无需手动调整。

### C. 动画说明

**SettingsDrawer 宽度动画**：
- 关闭：`width: 0`，整个抽屉隐藏（`overflow: hidden` 削剪内容）
- 打开：`width: 20rem`，抽屉占据右侧空间
- 主对话区：自动因 flex-1 被压缩，但内容自适应
- 滚动条：自然落在主区域与设置区交界处

**无边界效果**：
- TopBar 无 `border-b`
- ConversationArea 无 `border-r` 或移除条件
- SettingsDrawer 无 `border-l`
- 整体布局依靠 flex 间距和背景色差异区分区域

### D. 后续改进方向

1. **TopBar 可插拔插槽**：支持添加额外的全局控件（通知、用户菜单等）
2. **SettingsDrawer 宽度自适应**：在移动设备上调整宽度
3. **主题过渡动画**：针对 isDark 的 CSS 变量平滑过渡
4. **快捷键支持**：Ctrl/Cmd + K 打开设置（可选）
