## 🎯 关键概念速查

### 权限模式（Permission Mode）

| 模式 | 含义 | 执行策略 |
|------|------|---------|
| **default** | 默认 | 每次操作都询问用户 |
| **dontAsk** | 不再询问 | 已知操作不再询问 |
| **plan** | 计划模式 | 仅预览，不执行 |
| **acceptEdits** | 自动接受 | 自动接受所有编辑 |
| **bypassPermissions** | 绕过 | 绕过所有权限检查 |

### 权限行为（Permission Behavior）

| 行为 | 含义 | 触发场景 |
|------|------|---------|
| **allow** | 允许 | 规则明确允许或自动模式 |
| **deny** | 拒绝 | 规则明确禁止或危险操作 |
| **ask** | 询问 | 默认模式或不确定操作 |

### 危险文件保护

```
.bashrc, .zshrc, .gitconfig, .ssh/config,
.mcp.json, .claude.json, ...
```

### 危险目录保护

